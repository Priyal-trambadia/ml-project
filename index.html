Machine Learning Algorithms

We evaluated multiple regression models to ensure the highest accuracy for score prediction.

Recommended

Random Forest Regressor

How it works: Constructs hundreds of "Decision Trees" during training and averages their outputs.

Why we use it: It provides the highest accuracy by reducing overfitting. It handles mixed data exceptionally well.

Baseline

Linear Regression

How it works: Fits a straight line through the data points to establish a direct relationship between input variables and the Target score.

Use Case: Great for understanding simple correlations, such as "1 extra hour of study adds 2 points to the score."

Advanced

Gradient Boosting (XGBoost)

How it works: Builds trees sequentially, where each new tree specifically tries to correct the errors made by the previous ones.

Use Case: Often wins data science competitions due to high precision, though it requires more tuning than Random Forest.

Simple alternative

Decision Tree Regressor

How it works: Splits data into branches like a flowchart based on conditions (e.g., "Is Attendance > 80%?").

Use Case: Very easy to visualize and explain, but can be unstable on its own compared to the Random Forest ensemble.
